## 🎯 v0.8.0 开发规划（Web 管理控制台 - 基础框架）

### 目标
开发基于 Web 的管理控制台，提供可视化的系统管理功能。第一阶段实现基础框架和用户登录认证。

### 技术栈
- **前端**: Vue 3 + Vite + Element Plus + TypeScript + Pinia
- **后端**: 复用现有 HTTP API，新增管理端专用 API
- **认证**: JWT Token（复用 `src/auth/jwt.rs`）

### 详细计划
详见：`docs/admin-dashboard-plan.md`
